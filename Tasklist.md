# WordFlow Tasklist

## 1. 작업 원칙

- 이 문서는 `PRD.md`, `README.md`, `UI.md`를 기준으로 한 구현 작업 분해 문서다.
- 태스크는 최소 함수 단위까지 쪼개고, 함수 단위 테스트가 통과하면 상위 태스크를 완료 처리한다.
- 완료 체크는 실제 코드 반영과 테스트 통과가 확인된 경우에만 한다.
- 모든 커밋 메시지는 한국어로 작성한다.
- 현재 문서 개편 기준으로 아래 태스크는 모두 미완료 상태에서 시작한다.

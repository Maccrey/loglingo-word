# UI 디자인 가이드 (Paper-like Aesthetic)

## 0. 디자인 시각 자료 및 테마
- **컨셉**: 아날로그 종이, 잉크, 형광펜을 모티브로 한 편안하고 따뜻한 언어 학습 환경
- **전역 CSS 변수 (`globals.css`) 기반**:
  - 배경: `--bg-paper` (#fdfbf7), `--bg-card` (#ffffff)
  - 텍스트/선: `--text-ink` (#2c2a25), `--text-faded` (#7a756d), `--border-pencil` (#e8e3d8)
  - 그림자: `--shadow-paper`, `--shadow-card` (종이가 겹친 듯한 얕고 부드러운 그림자)
  - 강조 색상 (파스텔톤 형광펜 느낌): `--accent-yellow`, `--accent-green`, `--accent-blue`, `--accent-pink`, `--accent-orange`
  - 액션 버튼: `--btn-primary-bg` (#e8b066)

## 1. UI 구조
### 1.1 온보딩
- 언어 선택 (모국어, 학습 언어)
- 목표 설정
- 첫 학습 시작

### 1.2 홈 대시보드
- 오늘의 학습 요약 (종이 질감 카드 형태, 부드러운 그림자)
- 연속 학습(streak) 및 포인트(point) 표시 패널
- 빠른 시작 (Quick Start) 액션 버튼

### 1.3 플래시카드 학습
- 카드뷰 패널 기반 UI (카드 겹침 및 뒤집힘 효과 연출)
- 앞면: 단어 제시
- 뒷면: 의미 확인 및 예문
- 난이도 버튼 (형광펜 포인트 컬러 활용): Easy, Normal, Hard

### 1.4 퀴즈
- 객관식 선택형 문항 
- 주관식 짧은 타이핑 입력창 (스케치된 테두리 및 오답 시 붉은 잉크 느낌 피드백)

### 1.5 문장 생성 UI
- `badge` 형태의 단어 블록 조합 버튼
- 클릭 및 조합 인터페이스
- 완성된 문장 검증 및 즉각 피드백 제공

### 1.6 AI 대화 (채팅)
- 말풍선형 UI (`사용자`: 파스텔 블루, `AI`: 파스텔 옐로우, `교정`: 파스텔 그린 보드)
- 부드러운 입력 가능한 텍스트 공간 (textarea)
- 교정 사항 및 문법 설명 인라인 표시

### 1.7 리더보드
- 순위별 카드 리스트 구조
- 현재 사용자(나)의 위치 강조 (배경 하이라이트 색상 활용)
- 주간별 순위 및 획득 포인트 표시

### 1.8 SNS 피드
- 다른 사용자의 학습 성과 결과물 공유 카드
- 좋아요 인터랙션 (활성화 시 핑크색 강조)
- 공유 버튼 및 포인트 획득 알림 (StatusMessage 배너 활용)

### 1.9 설정
- 언어 설정, 알림 설정 토글 스위치 및 버튼
- 구독형 결제 (프리미엄) 구매 버튼 및 Entitlements 정보 안내 카드

## 2. UX 핵심 원칙
- **시각적 편안함**: 장시간 학습에도 눈이 피로하지 않도록 텍스트 명도를 낮추고 형광 네온 컬러를 은은한 파스텔 채도로 조절
- **1초 내 반응**: 모든 인터랙션은 즉각적인 피드백(버튼 로딩상태 / 뱃지 변화)을 렌더링
- **1분 내 성취 제공**: 짧은 퀴즈 및 플래시카드 단위로도 명확한 보상을 시각화하여 도달률 확인
- **긍정적 보상**: 과하지 않은 포인트 컬러로 강화를 제공 (예: 정답 및 콤보 획득 시 `--accent-green`)

## 3. UI 상태 관리 (StatusMessage)
피드백 메세지는 아날로그 테마에 맞추어 다음 3가지 톤으로 관리:
- **Loading (대기/진행)**: `--accent-yellow` 바탕, 잉크 텍스트
- **Success (성공/정답)**: `--accent-green` 바탕, 딥 그린 텍스트 (#2d7a4d)
- **Error (오류/오답)**: `--accent-pink` 바탕, 붉은 텍스트 (#d32f2f)

## 4. 반응형 레이아웃
- 컴포넌트 간격 및 여백은 `gap`, `padding` 비율 기반으로 유연하게 설정
- **Mobile First**: 모바일 단일 컬럼 뷰포트 최우선 대응
- **Tablet / Desktop**: 넓은 그리드 폭(`max-width: 980px` 내외)을 활용하여 패널 및 카드 레이아웃이 자연스럽게 확장되도록 지원
